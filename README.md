# 🎙️ Khak AI Voice Studio

**Khak AI Voice Studio** aik muft aur unlimited AI Text-to-Speech (TTS) tool hai jo aapke likhay huay alfaz ko bilkul qudrati (Natural) insani awaz mein badal deta hai. Ye tool khas tor par Content Creators, YouTubers, aur Educators ke liye banaya gaya hai.

## 🚀 Live Demo
Aap is app ko yahan live check kar sakte hain:
👉 [khak-ai-voice-studio.streamlit.app](https://khak-ai-voice-studio.streamlit.app/)

---

## ✨ App ke Khusoosiyat (Key Features)

* **Unlimited Usage:** Kisi qism ki character limit nahi hai. Jitni bari script ho, aap muft voice generate kar sakte hain.
* **100+ Global Languages:** Urdu 🇵🇰, Hindi 🇮🇳, English 🇺🇸, Arabic 🇸🇦, aur mazeed 100 se zayda zabanon ki support.
* **Natural AI Characters:** Is mein professional male aur female characters shamil hain jo bilkul real lagte hain.
* **Customization:** Aap awaz ki **Pitch** (bhari ya barik) aur **Speed** (tez ya ahista) ko apni marzi se adjust kar sakte hain.
* **High Quality Downloads:** Voice generate hone ke baad aap asani se MP3 format mein download kar sakte hain.
* **No Login Required:** App use karne ke liye kisi account ya login ki zaroorat nahi.

## 📖 Kaise Istemal Karen? (How to Use)
1.  **Language Select Karen:** "Select Country/Language" se apni pasand ki zaban chunein.
2.  **Character Chunein:** List mein se koi bhi male ya female character select karen.
3.  **Script Likhen:** Script box mein apna text paste karen.
4.  **Generate Karen:** "Generate Voice" button par click karen aur kuch hi seconds mein apni audio hasil karen.
